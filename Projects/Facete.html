Facete

http://aksw.org/Projects/Facete
Abstract:
The Facete project is comprised of a JavaScript library for faceted browsing of RDF data and (at the moment) an application for browsing geo-related RDF data. The application thereby offers filtering by facets, pivoting, and display of the data on a map.
