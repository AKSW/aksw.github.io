NLP2RDF

http://aksw.org/Projects/NLP2RDF
Abstract:
NLP2RDF is a LOD2 Community project that is developing the NLP Interchange Format (NIF). NIF aims to achieve interoperability between Natural Language Processing (NLP) tools, language resources and annotations. The output of NLP tools can be converted into RDF and used in the LOD2 Stack. The latest version and up-to-date information can be found on the [NLP2RDF landing page](http://persistence.uni-leipzig.org/nlp2rdf/ "http://persistence.uni-leipzig.org/nlp2rdf/").
