jena-sparql-api

http://aksw.org/Projects/jena-sparql-api
Abstract:
A java library that offers several Jena-compatible ways to transparently add delays, caching, pagination, retry and even query transformations before sending off your original SPARQL query. Also, the server module bundles Jena with the Atmosphere framework, giving you a kickstart for REST and websocket implementations.
