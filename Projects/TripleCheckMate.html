TripleCheckMate

http://aksw.org/Projects/TripleCheckMate
Abstract:
TripleCheckMate is a tool for crowdsourcing the assessment of Linked Open Data. It was developed under the DBpediaDQ project as an application for evaluating the correctness of DBpedia. TripleCheckMate provides an easy-to-use user interface with multiple resource assignment methods and a ready to use error classification scheme. It is released under Apache 2.0 Licence.
