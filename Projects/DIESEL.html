DIESEL

Distributed Search in Large Enterprise Data

Abstract:
What we want to do: Develop a generic keyword search and question answering infrastructure for distributed and structured enterprise data. Our framework will exploit the distribution of the data to improve both the interpretation and the federated execution of user queries while abiding by the users’ access restrictions. We will deliver an open-source version of the framework that implements keyword search functionality as well as prototypical extensions of the partners’ product suites and use case studies.
